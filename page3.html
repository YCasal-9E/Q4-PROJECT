Quarter 2 Project

Responsive Design

We explored how to create websites that adapt to different screen sizes using responsive design principles. This included using flexible grids, fluid images, and media queries to ensure our layouts work well on desktops, tablets, and mobile devices.

CSS Frameworks

This section introduced us to Bootstrap, a popular CSS framework. We learned how to use its pre-built classes for layout, typography, buttons, navigation bars, and components to build stylish and mobile-first web pages efficiently.

In this quarter, we focused on making our websites more professional and user-friendly. We learned how to design for all screen sizes through responsive layouts, and how to speed up development using Bootstrap’s powerful and ready-to-use components.
