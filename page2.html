Quarter 1 Project

HTML Structure

We learned about the foundational structure of a webpage using HTML. This included understanding tags like <html>, <head>, <body>, and other semantic elements like headings, paragraphs, lists, and images.

CSS Styling

This part focused on designing web pages using CSS. We explored inline, internal, and external CSS, learned about selectors, properties, and values, and applied styles to improve layout, typography, and color schemes.

In this quarter, we explored the basics of front-end development. We were introduced to how websites are structured with HTML and how to make them visually appealing using CSS. This foundation helped us understand how content and design work together to create functional web pages.
